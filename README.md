# 图书管理系统 python340

本项目是一款基于Python开发的图书管理系统，包含图形用户界面（GUI），操作便捷，支持在PyCharm环境中直接运行。

## 技术栈

- 后端：Python
- 前端：PyQt5（用于构建GUI）

## 功能模块

### 普通用户

- 登录
- 查阅图书信息
- 借书
- 还书

### 管理员

- 录入图书
- 删除图书
- 统计书籍信息

## 系统角色

- 普通用户：可查阅图书信息、借阅和归还图书
- 管理员：拥有图书录入、删除及信息统计权限

## 运行环境

- 操作系统：支持Windows、Linux、MacOS
- Python版本：3.6及以上

## 部署步骤

1. 安装Python（版本3.6及以上）
2. 安装PyCharm或其他支持Python的IDE
3. 下载项目源码及素材
4. 在PyCharm中打开项目并运行
